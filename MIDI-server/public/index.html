MIDISERVER

// COLLABORATIVE NETWORK MIDI RELAY SERVER

// GETTING STARTED
  1. Click [ ENTER ] on the splash screen to unlock audio
  2. Enter a room name and click [ JOIN ]
  3. Share the room name with collaborators — all peers sync automatically
  4. Draw notes on the piano roll, set BPM, and press ▶ PLAY
// TRANSPORT
▶ PLAY / ■ STOP Start and stop synchronized playback across all peers
♩ CLICK Toggle the metronome click track (downbeat accent on beat 1)
BPM Drag the slider to change tempo — broadcasts to all peers
// PIANO ROLL
Left-click empty Add a note, snapped to 1/16th beat
Left-click note Delete the note
Drag note Move note (pitch and/or beat position)
Right-click note Delete the note
Scroll wheel Scroll pitch range (vertical)
Shift + scroll Scroll time (horizontal)
Ctrl + scroll Zoom horizontal
↓ DROP .MID Drag a .mid file onto the zone to import it
↓ EXPORT .MID Download the current roll as a MIDI file
PR ON Enable piano roll playback (routes to synth and/or MIDI output)
BARS Pattern length in bars (1 / 2 / 4 / 8)
CH / VEL Channel and velocity for new notes
// MIDI I/O
INPUT / OUTPUT Select hardware MIDI devices (requires browser MIDI permission)
THRU Pass MIDI input through to the output device
CLK ↑ Relay MIDI clock (0xF8) messages to the output device
PANIC Send all-notes-off to the output device
FILTER CH Only pass through MIDI on a specific channel (or all)
REMAP → Remap incoming MIDI to a different output channel
// MONITOR
CC Bar graph per controller number, labeled with CC name and value
PROGRAM CHANGE Last program received per channel with GM patch name
PITCH BEND Center-anchored bar per channel showing bend amount
// MIDI RECORDING
● REC Start recording all incoming MIDI events
■ STOP Stop recording
↓ EXPORT .MID Download the recording as a .mid file
// ROOMS

Connect multiple browser tabs or devices to the same room by entering the same room name. All play/stop commands, BPM changes, and piano roll state broadcast to all peers. Your latency is shown in the header as you Xms.
