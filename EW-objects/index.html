// DISTRIBUTED MODULAR AUDIO //

Each object is a standalone audio module connected via WebSocket rooms. Room URLs are patch cables. Paste one module's room name into another module's input to connect them.

Modules can live in different browser tabs, machines, or be shared between users in real-time.

// HOW TO CONNECT //

1. Open an object (e.g. Oscillator) in a browser tab. The page auto-generates a room name and displays it in the OUTPUT ROOM panel.

2. Open another object in a second tab. In the INPUT CONNECTIONS panel, paste the first object's room name into the desired input port.

3. Click LINK. The second object now receives real-time parameter updates from the first.

4. Changes to the source object's controls are instantly reflected in all connected subscribers.
